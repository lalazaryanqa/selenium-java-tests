# Selenium UI Tests

Java-based UI test automation project using Selenium WebDriver and TestNG.

## Overview

This project contains automated UI tests created for Selenium WebDriver practice and QA Automation training.  
The framework demonstrates basic UI automation concepts, test structure organization, and Page Object Model (POM) implementation.

---

## Technologies

- Java
- Selenium WebDriver
- TestNG
- Gradle
- Page Object Model (POM)
