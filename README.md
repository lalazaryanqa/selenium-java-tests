# Selenium UI Tests

UI test automation framework built with Selenium WebDriver and TestNG.

## Overview

This project demonstrates automated UI testing using the Page Object Model (POM) design pattern. The framework includes structured test data management through DTO objects and Enums, reusable page components, and validation using TestNG assertions.

The main test scenario automates the DemoQA Practice Form and validates submitted user information.

## Technologies

* Java
* Selenium WebDriver
* TestNG
* Gradle
* Page Object Model (POM)

## Features

* Automated UI testing
* Page Object Model architecture
* DTO-based test data management
* Enums for reusable test data
* CSS and XPath locator practice
* Soft Assertions
* Form validation testing
* Reusable page components

## Project Structure

* dto
* enums
* manager
* pages
* tests

## Test Scenario

The project automates the DemoQA Practice Form:

* Fill user data
* Select gender
* Select hobbies
* Choose state and city
* Submit the form
* Validate confirmation modal
* Verify submitted values
